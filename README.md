# ⚡ PromptVault

> AI Prompt Templates That Actually Work. 500+ curated prompts for writing, coding, marketing, and more.

## Features

- 🔍 Search & filter by category
- 📋 One-click copy to clipboard
- 🏷️ Tagged by model (ChatGPT, Claude, Gemini)
- 💎 Free + Pro tiers ($14.9/mo)
- 📱 Responsive design

## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS
- Stripe (payments - coming soon)
- Vercel (deployment)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
